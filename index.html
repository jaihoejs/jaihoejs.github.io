Welcome to Jaihoe

Jaihoe is a JavaScript library that is built on top of JavaScript for running JavaScript code in Jaihoe Syntax.

Jaihoe syntax is called JaihoeScript which is a manifesting programming language to build HTML/JS applications in HTML Syntax.

Feature and Limitation

Dependencies

Add different dependency based on your code project. Depending on the state of the code.

HTML Syntax

It uses HTML syntax with HTML attributes along with nested HTML blocks.

Strictly Typed Attribute

You cannot change the arrangement of the attributes in a HTML block only the blocks of HTML.

External Execution

You can make another HTML file for the script execution of Jaihoe. In that way, maximize code on the other file

Recent Updates

Jan 16, 2023

Third Update
Program Compile Update and Fun

Before Jaihow can read and perform one block at a time so there is a heirarcy, now it can read and perform multiple block at the same time.

Additionally, you can now do some wizardly tricks like bliking text, rainbow text, scrolling title bar, and etc by using (jaihoe_fun) as a dependency.

January 15, 2022

Second Update
Curriencies

You can now convert some currencies of the countries with each other vice versa by using (jaihoe_currency) as a dependency.

Countries that are included were United States, Australia, Korea, Japan, Brazil, China, Columbia, Mexico, Philippines, Russia, Singapore, Hong Kong, Thailand, Indonesia, India and etc.

January 14, 2021

First Update
Unit Conversions

You can now do some conversion of volume, weight, temperature, energy, area and etc. by using (jaihoe_convert) as a dependency.

It also have other conversion features like rounding, with unit, with tip, with number seperation and etc.
